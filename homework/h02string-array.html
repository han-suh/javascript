문자열과 배열 실습

  1. "안녕"=="안녕" : true 둘 다 문자열로 기본형이며 같은 값을 가짐
  2. "안녕"==new String("안녕") : true 기본형과 자료형이지만 같은 값을 가지고 있기 때문에 js에서 같은 값이면 기본데이터로 정의
  3. "안녕"===new String("안녕") : false 문자열로 같은 값이지만 기본형과 자료형으로 데이터가 완전히 같지 않기 때문에 false
  4. ['안','녕']==['안','녕'] : false 자료형의 경우 비교할 수 없기 때문에 비교연산이 불가
  5. new String("안녕")==new String("안녕") : 랩퍼클래스의 자료형은 값을 참조하기 때문에 값을 비교해서 true여야하지만 결과는 false, 동등비교연산에 값이 존재해야 자료형에서 값을 찾을 수 있다.
  6. 사과,배,자두,딸기 를 순서대로 저장하는 배열 자료를 생성
  7. 위 자료에 index가 2인 자료를 출력
  8. 위 자료에 '수박'을 추가
  9. 위 자료의 길이를 출력
  10. typeof "안녕" : string
  11. typeof new String("안녕") : object
