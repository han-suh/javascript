prompt로 수 2개를 입력받아서 두 수의 곱한 값을 alret으로 출력하세요.

  • 입력과 출력에 대한 이해
  • 2개를 입력 받아서 변수에 참조
  • 데이터의 종류와 형변환 (문자열->숫자)
  • 연산에 대한 이해
prompt로 수 두 개를 입력받아서 곱하기 연산을 하라 let x=prompt("수를 입력"); //"39"(문자열) -> 39 let y=prompt("수를 입력"); x=Number(x) y=Number(y) alert(x*y) // 자바스크립트는 곱하기 연산을 수로 먼저하려고 함 - x=Number(x) 과정 없어도 가능
