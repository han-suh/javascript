Array의 유용한 method

Array는 자주 사용되는 자료이기 때문에 유용한 함수를 제공하고 있다.

요소를 추가하는 메서드

  • push(data) : 배열의 맨 끝에 요소 추가
  • unshift(data) : 배열의 맨 앞에 요소 추가(모든 요소를 뒤로 이동 => 부자연스러움)
  • pop(data) : 배열의 맨 끝에 요소 제거
  • shift(data) : 배열의 맨 앞에 요소 제거(모든 요소를 앞으로 이동 => 부자연스러움)

splice는 배열에서 특정 요소를 삭제하고 배열로 반환하는 메소드

  • arr.splice(startIndex) : 배열의 시작점부터 끝까지 삭제하고 배열을 반환.
  • arr.splice(start,count) : 배열의 시작점부터 count까지 삭제하고 배열을 반환
  • arr.splice(start,count,...args) : 배열의 시작점부터 count까지 삭제하고 배열을 반환한 후 args 매개변수를 시작점부터 추가
  • startIndex의 음수 : 끝점부터 시점

concat 배열의 + 연산 (지원하는 프로그램언어가 많지 않다.)

  • new Array().concat([...],[...],[...]...) : 앞쪽 배열을 기준으로 그 다음 배열을 더하는 연산
  • [...].concat([...],...) : 앞쪽 배열을 기준으로 그 다음 배열을 더하는 연산
  • "안녕".concat("!") == "안녕"+"!" : 두 문자열을 더해서 새로운 문자열을 만드는 연산
  • 기본형은 불변데이터이기 때문에 기존의 데이터를 변경하지 않는다.
  • Array 자료형은 길이가 변경 가능한 배열자료(시퀀스)다.

배열의 상태를 문자열로 출력하는 toString(),join("구분자")

  • toString() : 배열 내부의 아이템을 구분자 ","로 결합해서 문자열로 반환
  • join("구분자") : toString()과 같은데 구분자를 변경 가능

특정 인덱스의 아이템을 참조하는 at()

at()은 []과 같이 index의 아이템을 참조하고 -1로 끝쪽 아이템을 참조할 수 있다.
