배열(list)과 문자열(string)

순서가(seq) 있는 자료 배열 Array

  • Array : 순서대로 데이터를 참조하는 자료형
  • 선언법1 : new Array()
  • 선언법2 : let arr=["사과", "딸기", "배", 1, true, new Date()...]
  • 특정 순서(index)의 자료를 참조하는 법 : arr[0~n]
  • 자료의 길이 : arr.length
  • 자료를 뒤쪽에 추가 : arr.push(data)
  • 특정 위치에 자료를 추가하거나 수정하는 것 : arr[0~n]=data

순서가 있는 자료인데 문자로만 된 기본형 데이터 String

  • String : 문자로만 된 순서가 있는 데이터로 js에서 기본형 데이터로 정의해서 사용중
  • 선언법 : let s="안녕"
  • 기본형 데이터 랩퍼 클래스 String : 문자열의 처리를 도와주는 함수를 지원, 존재하지 않는 필드를 참조하도록 도와준다.
  • 문자열 데이터를 많이 사용해서 js는 기본형으로 취급합니다.
  • Array와 아주 비슷하기 때문에 유사한 필드나 함수를 지원한다.
