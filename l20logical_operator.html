논리 연산자로 복잡한 제어문을 작성할 수 있게하자

논리연산은 비트연산과 비슷하다.

  • || Or : 논리의 합
    • true || true : true
    • true || false : true
    • false || true : true
    • false || false : false
    • **논리 합은 한 개라도 true면 모두 true다
  • && And : 논리의 곱
    • true && true : true
    • true && false : false
    • false && true : false
    • false && false : false
    • **논리 곱은 한 개라도 false면 모두 false다

논리연산과 중단 점(논리연산이 멈추는 지점)

js 엔진은 논리연산에서 참(||) 또는 거짓(&&)이 나올거라는 확신이 있다면 다음을 하지 않는다.

논리연산의 우선 순위

() >> 곱하기 > 더하기

a || b && c || b && d => {a || (b && c) || (b && d)}

a && b || c && b || d => {(a && b) || (c && b) || d}

true || true && false || true && false => true || false || false => true

true && false || true && true || false => false || true || false => true
